## 💡 About the Project

**TravelDost** (also known as *Videsh Sathi*) is designed to solve a critical problem for tourists: **Uncertainty in a new environment.**

Instead of searching Google for "police numbers in [City]" or "tourist rules in [Region]," TravelDost automatically detects where the user is and instantly configures the dashboard to show:
1.  **Emergency SOS contacts** (Police, Ambulance, Women's Helpline).
2.  **Local "Uncommon Rules"** to avoid fines and cultural misunderstandings.
3.  **Language utilities** like translation cards and currency converters.

This repository contains the **Frontend Implementation** using the React + Vite ecosystem.

---

## 🚀 Key Features

* **📍 Smart Location Detection:** Automatically identifies the user's city/region to fetch relevant data.
* **🚨 Instant SOS:** One-tap access to local emergency services (Police: 100, Ambulance: 108, etc.).
* **📜 Uncommon Rules:** A dedicated section for local laws and cultural "Do's and Don'ts."
* **🗣️ Language Tools:** Built-in basic translator and currency converter widgets.
* **✅ Trip Checklist:** Interactive to-do list for managing travel documents and essentials.
* **🗺️ Interactive Map:** Visualizes the user's location and nearby points of interest.

## 🏁 Getting Started

Follow these steps to set up the project locally on your machine.

### Prerequisites
* Node.js (v18 or higher)
* npm or yarn

### Installation

1.  **Clone the repository**
    ```bash
    git clone [https://github.com/your-username/TravelDost.git](https://github.com/your-username/TravelDost.git)
    cd TravelDost
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Run the development server**
    ```bash
    npm run dev
    ```

4.  **Open your browser**
    Navigate to `http://localhost:5173` to see the app in action.
